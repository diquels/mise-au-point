# UI Kit : Instagram Carousel

The single "product" surface for Pierre Diquelou Photographie : carrousels Instagram pédagogiques (1080×1350).

## Components

- `Slide.jsx` : wrapper 1080×1350, gère le fond (Layout A bleu / Layout B cuivre) + vague + boussole + handle.
- `Title.jsx` : titre serif Cormorant, doré sur bleu / bleu sur cuivre.
- `Pill.jsx` : étiquette ovale italique « en 3 gestes… ».
- `Body.jsx` : paragraphe Inter justifié.
- `SocialIcons.jsx` : fragment image cœur/bulle/partage/signet.
- `Curve.jsx` : courbe ornementale en coin.

## Demo

`index.html` montre un carrousel complet de 4 slides, navigable par flèches ←/→ ou clic.
